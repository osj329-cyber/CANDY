## Active Projects

Use this section to keep track of projects you are currently working on.

When starting something new, say:

> Add this to Active Projects.

When a project is finished, say:

> Archive this project.

Each project should include the project name, current status, and next action.

Example:

- [Project name] — [Current status]. Next step: [next action].
